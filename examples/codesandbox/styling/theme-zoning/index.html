Theming and CSS scoping

The color scheme of carbon-custom-elements components can be changed either by CSS custom properties (this example) or by creating inherited components with the static styles property changed.

If you inspect the style of <footer> in this example, you'll find some Carbon theme custom properties there are changed to ones of g100 theme, and see the color of <bx-btn> reflects such change.

As Shadow DOM (one of the Web Components specs that carbon-custom-elements uses) promises, styles that carbon-custom-elements defines does not affect styles in your application.
