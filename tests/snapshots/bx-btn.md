# `bx-btn`

## `Misc attributes`

####   `should render with minimum attributes for <button>`

```
<button
  class="bx--btn bx--btn--primary"
  id="button"
>
  <slot>
  </slot>
  <slot name="icon">
  </slot>
</button>

```

####   `should render with various attributes for <button>`

```
<button
  autofocus=""
  class="bx--btn bx--btn--disabled bx--btn--secondary bx--btn--sm"
  disabled=""
  id="button"
  type="submit"
>
  <slot>
  </slot>
  <slot name="icon">
  </slot>
</button>

```
